- 用闭包来构建
    函数内嵌函数 就会形成作用域的嵌套，形成闭包
    因为闭包内部的函数会保存对外部函数内的变量的持久引用，让这些变量的生命周期延长
    函数在运行时，可以引用到定义时的上下文环境内的变量。
